LibConfig

The most important interface used to configure components is LibConfig. In case of plain-gallery and carousel you have to pass an object of type LibConfig as component input. Instead, in case of modal-gallery it will be a parameter of a service function.

LibConfig has only optional properties. When any of them is undefined, a default value will be applied, as described here.

LibConfig is an object with many optional properties. However, not all of them are used by all components. Here you can find a table to know which of these properties can impact on components

All these properties have spefic roles

slideConfig
Type: SlideConfigoptional

Optional object used to configure SlideConfig to manage sliding features of modal-gallery and ....
accessibilityConfig
Type: AccessibilityConfigoptional

Optional object used to configure AccessibilityConfig.
previewConfig
Type: PreviewConfigoptional

Optional object used to configure PreviewConfig.
buttonsConfig
Type: ButtonsConfigoptional

Optional object used to configure ButtonsConfig.
dotsConfig
Type: DotsConfigoptional

Optional object used to configure DotsConfig.
plainGalleryConfig
Type: PlainGalleryConfigoptional

Optional object used to configure PlainGalleryConfig.
currentImageConfig
Type: CurrentImageConfigoptional

Optional object used to configure CurrentImageConfig.
keyboardConfig
Type: KeyboardConfigoptional

Optional object used to configure KeyboardConfig.
carouselConfig
Type: CarouselConfigoptional

Optional object used to configure CarouselConfig. Used only by carousel.
carouselImageConfig
Type: CarouselImageConfigoptional

Optional object used to configure CarouselImageConfig. Used only by carousel.
carouselPreviewsConfig
Type: CarouselPreviewConfigoptional

Optional object used to configure CarouselPreviewConfig. Used only by carousel.
carouselPlayConfig
Type: PlayConfigoptional

Optional object used to configure PlayConfig. Used only by carousel.
carouselDotsConfig
Type: DotsConfigoptional

Optional object used to configure DotsConfig. Used only by carousel.
keyboardServiceConfig
Type: KeyboardServiceConfigoptional

Optional object used to configure KeyboardServiceConfig. Used to pass custom configuration to mousetrap to download images with keyboard shortcut, and also to disable SSR in case you want to use SystemJS.
enableCloseOutside
Type: booleanoptional

Optional property to close modal-gallery clicking on the semi-transparent background.
